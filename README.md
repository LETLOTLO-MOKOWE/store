## FRESHMART LOCAL GROCERY STORE WEBSIDE

## Website Description
This project is a website developed for FreshMart Local Grocery Store.The website provides
information about thr store,products and it allows customers to make enquiries online.

## Features 
-Home Page
-About Us Page
-Product Catalog
-Enquiry form 
-Contact page

## installation/Technologies Used 
-HTML

## How to Run
1.Open the project folder in visual studio code
2.Open index.html in your browser
